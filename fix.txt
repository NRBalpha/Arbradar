Fix 2 - Add guaranteed profit validation to arb pairs:
For each arb pair calculate:
yes_price = prob_low / 100
no_price = 1 - (prob_high / 100)
sum_prices = yes_price + no_price
If sum_prices >= 1.0:
  is_real_arb = False
  guaranteed_profit_pct = 0
Else:
  is_real_arb = True
  guaranteed_profit_pct = (1 - sum_prices) * 100
Add both fields to every arb pair response.
Only keep pairs where is_real_arb is True.
